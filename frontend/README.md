# NovaOps Dashboard (Phase 3)

Next.js dashboard for the NovaOps DevOps command center.

## Planned Features

- **Real-time Service Health** — Live health status for all monitored services
- **Metrics Dashboard** — CPU, memory, latency, and throughput charts
- **Commander Chat** — Interactive chat interface to the Commander Agent
- **Incident Timeline** — Historical view of incidents and resolutions
- **Deployment Tracker** — Track deployments across environments

## Tech Stack (Planned)

- Next.js 14+ (App Router)
- Tailwind CSS
- shadcn/ui components
- WebSocket for real-time updates
- Vercel deployment
